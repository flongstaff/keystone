# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is available at the link above. In short: be respectful, be constructive, and assume good intent.

## Enforcement

Instances of unacceptable behaviour can be reported by opening a GitHub issue or contacting the maintainer directly. All reports will be reviewed and handled appropriately.
